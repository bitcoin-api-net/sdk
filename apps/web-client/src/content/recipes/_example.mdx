A minimal example showing how to call the [`getCurrentPrice`](/docs/api/getCurrentPrice) endpoint and print the latest BTC/USDT price.

## Code

```js
const response = await fetch(
  'https://api.bitcoinapi.dev/v1/prices/current?symbol=btcusdt',
  { headers: { Authorization: 'Bearer YOUR_API_KEY' } }
);
const data = await response.json();
console.log(`BTC/USDT: ${data.price} @ ${data.time}`);
```

## Notes

- Replace `YOUR_API_KEY` with the key from your [profile](/profile/home).
- Symbols are lowercase. Currently only `btcusdt` is supported.
